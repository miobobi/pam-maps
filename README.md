# Google Maps – React Native (Expo)

Aplicativo desenvolvido para a atividade de integração do Google Maps.  
Inclui mapa em área segura, campo de busca para localizar endereços e uso de `.env` para proteger a API Key.  
Tecnologias: Expo, React Native Maps, Geocoding API e react-native-dotenv.

## Como executar:
1. Instale dependências: `npm install`
2. Crie `.env` com: `GOOGLE_MAPS_API_KEY=SUA_CHAVE`
3. Inicie com: `npx expo start`
